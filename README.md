# Trader Performance Analysis Based on Bitcoin Market Sentiment

## Project Overview
This project analyzes the relationship between Bitcoin market sentiment and trader performance using Bitcoin Fear & Greed sentiment data and historical trader data from Hyperliquid.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Excel

## Analysis Performed
- Data cleaning and preprocessing
- Date conversion and dataset merging
- Sentiment-wise PnL analysis
- Win rate analysis
- Buy/Sell side analysis
- Symbol-wise analysis
- Account-wise analysis
- Leverage analysis
- Daily PnL trend analysis

## Key Findings
- Highest average PnL was observed during Greed sentiment.
- Highest total PnL was observed during Fear sentiment.
- Highest win rate was observed during Fear sentiment.
- Fear sentiment showed stronger overall performance.
- Greed sentiment showed higher average profit per trade.

## Files Included
- `primetrade_analysis.ipynb` - Main analysis notebook
- `analysis_results.xlsx` - Excel output file
- `final_report.pdf` - Final report with charts and insights

## Conclusion
Bitcoin market sentiment has a clear impact on trader performance. Fear sentiment showed better overall results based on total PnL and win rate, while Greed sentiment showed higher average PnL per trade.
